Доступные пакеты доната

Властелин - 2000 руб

  • /clear - очистить инвентарь
  • /kill - убить себя
  • /freegm - выдать другу креатив

Повелитель - 1000 руб

  • /saveinv - сохранить инвентарь
  • /inv list - список сохраненных инвентарей

Фантом - 899 руб

  • /ban-ip - заблокировать игрока по айпи
  • /topmoney - узнать топ по балансу
  • /setwarp - установить варп

Император - 650 руб

  • /ban - заблокировать игрока на сервере
  • /unbantime - разблокировать игрока
  • /skin - украсть скин у игрока

Легенда - 499 руб

  • /giveitem - выдать предмет
  • /bantime - забанить игрока на время
  • /cc - очистить чат

Визер - 289 руб

  • /gm (0/1/3) - креатив
  • /repair - починить предмет
  • /kick - выгнать игрока с сервера

Суприм - 139 руб

  • /itemid - узнать айди предмета
  • /xyz - узнать координаты
  • /jump - подпрыгнуть до небес

Воин - 69 руб
